## 📱 Quick Actions

### Test Locally
```bash
# Just open the file
start.bat          # Windows
bash start.sh      # Mac/Linux
# Or double-click index.html
```

### Deploy to Firebase
```bash
firebase deploy --only hosting
```

### Deploy to Netlify
```bash
netlify deploy --prod
```

### Deploy to Vercel
```bash
vercel --prod
```
